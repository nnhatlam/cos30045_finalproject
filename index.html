Dashboard

This webpage is designed to inform policymakers, researchers, the public and media about trends in illegal mobile phone use while driving in Australia.

Mobile phone use while driving is a major safety issue in Australia. Existing dashboards often present only basic trends.

They do not fully leverage the detailed enforcement data collected since 2023, which limits deeper insights into driver behavior, demographical differences, and the impact of AI cameras on road safety.

This version aims to improve the current Data Hub by allowing users to interact with the information more deeply and clearly.
